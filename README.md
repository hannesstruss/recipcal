Prepare
-------

    npm install

Run tests (watch)
-----------------

    npm install -g karma-cli
    karma start

Run tests (once)
----------------

    npm test

Run app
-------

    npm install -g gulp-cli
    gulp watch

Then serve the build directory (e.g. `python -m SimpleHTTPServer`)
